# バージョン履歴

[English version](CHANGELOG.md)

## v0.4.0 (2026-05-17)

- 画像を右クリック →「ビューアーで画像を開く」を追加
- 小さい画像を拡大して範囲選択したいときや、ページ上でうまく選択できない画像でも、ビューアーで開いてOCRできます

## v0.3.0 (2026-04-29)

- 画像ビューアーページ追加（ポップアップ「画像を開く」「クリップボードの画像を開く」）
- ビューアーでファイル選択 / ドラッグ&ドロップ / Ctrl+V 貼り付けに対応
- ビューアーで「範囲選択を開始」「画像全体をOCR」を選択可能
- file:// の権限設定なしでローカル画像をOCR可能に
- clipboardRead 権限追加
- 設定（settings.ts）・クリーニングルールの共通モジュール化
- ビューアーでもアラート表示・クリーニングルールを反映
- ポップアップUIにセパレーター追加、ボタン色分け（青=OCR / グレー=開く）
- メッセージ送信を拡張ページ/通常ページで分岐（アラート重複修正）

## v0.2.0 (2026-04-27)

- タイトな範囲選択でもOCRが動くよう隣接色パディングを追加
- 短辺の30%（最大50px）を端ピクセル引き伸ばしで余白追加
- 白背景・暗背景どちらでも安定して検出可能に
- OCR統合テスト追加（onnxruntime-node + sharp）

## v0.1.0 (2026-04-19)

- 初回リリース
- 範囲選択OCR（ドラッグで囲んでクリップボードにコピー）
- DEIM（テキスト領域検出）+ PARSeq（文字認識）+ XY-Cut（読み順解析）
- 完全オフライン（通信ゼロ）
- 3通りの起動方法（ツールバー / Alt+Shift+O / 右クリックメニュー）
- 正規表現クリーニングルール（カンマ除去、スペース整理など）
- 日英UI自動切替
- モデルキャッシュ（IndexedDB）

オフラインOCR — どこにも通信しない安全な範囲選択OCR

画面上の好きな場所をマウスで選択するだけで、テキストを読み取ってクリップボードにコピーします。
OCR エンジンには国立国会図書館 (NDL) が開発した NDLOCR の軽量版を採用しており、日本語テキストの認識に優れています。
すべての処理はあなたのブラウザ内で完結。インターネット接続は不要で、画像やテキストが外部サーバーに送信されることは一切ありません。

■ 主な特徴

・完全オフライン — データは一切外部に送信されません
・ワンクリックで使える — 範囲を選択するだけで自動コピー
・日本語テキストに特化 — 国立国会図書館の OCR 技術をベースにした高精度な認識
・3つの起動方法 — ツールバーアイコン / キーボードショートカット / 右クリックメニュー
・正規表現クリーニング — カンマ削除や不要なスペースの除去など、コピー前に自動整形。自分だけのルールを自由に追加でき、請求書の金額処理など実務に合わせたカスタマイズが可能
・多言語対応 — 日本語・英語の UI を自動切替

■ 使い方

1. 拡張機能のアイコンをクリック（またはショートカットキー）
2. OCR したい範囲をドラッグで選択
3. 数秒待つと認識結果が自動でクリップボードにコピーされます

💡 Chrome で画像や PDF ファイルを直接開いてスキャンすることもできます。
　 ローカルのファイルを Chrome にドラッグ＆ドロップして、そのまま範囲選択するだけです。

■ こんな場面で便利

・PDF や画像内のテキストをコピーしたいとき
・請求書や帳票の数値を素早く取り出したいとき
・スクリーンショットから文字を起こしたいとき
・社外秘の文書を外部サービスに送りたくないとき

■ プライバシーについて

SnapOCR Offline は外部サーバーとの通信を一切行いません。
OCR エンジン (ONNX Runtime Web) とモデルはすべて拡張機能に内蔵されており、ブラウザ内で完結して動作します。
個人情報や画像データが収集・送信されることはありません。

■ 技術仕様

・OCR エンジン: NDLOCR-Lite (国立国会図書館 OCR の軽量版)
・検出モデル: DEIM (FP32)
・認識モデル: PARSeq (FP32)
・推論ランタイム: ONNX Runtime Web (WebAssembly)
・対応言語: 日本語（横書き）
・動作環境: Chrome 116 以降

■ 謝辞

本拡張機能の OCR エンジンおよびモデルは、国立国会図書館 (NDL) が研究・開発・公開した NDLOCR をベースにしています。
高精度な日本語 OCR 技術をオープンに公開してくださった国立国会図書館に深く感謝いたします。

オフラインOCR — どこにも通信しない安全な範囲選択OCR（Chrome拡張機能）

画面上のテキストをマウスで選択するだけで、OCR認識してクリップボードに自動コピー。
すべての処理がブラウザ内で完結し、データが外部に送信されることは一切ありません。

▼ Chrome Web Store
https://chromewebstore.google.com/detail/offline-ocr/（公開後にURLを差し替え）

▼ GitHub
https://github.com/tamoco-mocomoco/offline-ocr

━━━━━━━━━━━━━━━━━━━━
📌 主な特徴
━━━━━━━━━━━━━━━━━━━━
🔒 完全オフライン — データは一切外部に送信されません
🇯🇵 日本語特化 — 国立国会図書館の OCR 技術（NDLOCR）ベース
📐 縦書き・横書き対応 — 古い書籍や法律文書にも対応
⚡ ワンクリック起動 — ツールバー / Alt+Shift+O / 右クリックメニュー
🧹 正規表現クリーニング — OCR結果をコピー前に自動整形

━━━━━━━━━━━━━━━━━━━━
📖 使い方
━━━━━━━━━━━━━━━━━━━━
1. 拡張機能のアイコンをクリック（またはAlt+Shift+O）
2. OCRしたい範囲をマウスでドラッグ
3. 数秒で認識結果が自動でクリップボードにコピー

━━━━━━━━━━━━━━━━━━━━
🔧 技術仕様
━━━━━━━━━━━━━━━━━━━━
・OCRエンジン: NDLOCR-Lite（国立国会図書館OCRの軽量版）
・検出モデル: DEIM (FP32)
・認識モデル: PARSeq (FP32)
・推論: ONNX Runtime Web (WebAssembly)
・動作環境: Chrome 116以降
